Minecraft Fabric Plugin

This project has a growing list of features, but some of the most important ones are:

Bright Celestial Objects API and Frontend

A RESTful API that returns the 20 brightest celestial objects (stars, planets, moons) visible above the horizon at any given location and time. Built with Python and FastAPI.

A static frontend is available for accessing the API with sky map and attribute explanations.

  • Real-time astronomical calculations using PyEphem
  • Support for 50+ celestial objects (8 planets, Sun, Moon, 4 Galilean moons, 35+ bright stars)
  • Comprehensive position data (altitude, azimuth, right ascension, declination, magnitude)
  • OpenAPI Specs (Swagger)

Tech: Python • FastAPI • PyEphem • Docker • JavaScript (Frontend)
